# MEAN Stack using Angular 2

1) Install packages
```console
npm install -g angular-cli
```



Created angular 2 and build
```console
cd /public/vendor
ng new my-app --prefix my-app --skip-git true
cd my-app
ng build -prod --output-path ./../../my-app
```
